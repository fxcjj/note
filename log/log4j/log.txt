Logger和Appender关联
	a) Logger1打印到console
	b) Logger2打印到file
	
Logger logger1 = new Logger("com.fool");
Logger logger2 = new Logger("com.smart");
logger1.addAppender(new FileAppender("d:/logs/fool.log"));
logger2.addAppender(new FileAppender("d:/logs/smart.log"));

SLF4J
Simple Logging Facade For Java

SLF4J是日志顶层抽象

实现有logback, log4j, java.util.logging, tinylog

logback是直接实现slf4j的
而log4j, java.util.logging, tinylog需要适配器

log4j需要引入的依赖
slf4j-api 
slf4j-log4j12 转换器
log4j

具体使用日志类的API：
1. log4j：
import org.apache.log4j.Logger;
Logger logger= Logger.getLogger(xx.class);
 
2. slf4j+log4j：（推荐）
import  org.slf4j.Logger;
import  org.slf4j.LoggerFactory;
Logger logger = LoggerFactory.getLogger(xx.class);
