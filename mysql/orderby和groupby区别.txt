 
ORDER BY和GROUP BY的用法

一、ORDER BY是按表中某字段排列表中数据
1、单字段排序 
例如:
	SELECT * FROM tn ORDER BY id DESC;（注DESC为降序，ASC或不写是升序） 
2、多字段排序，先按字段1排序，如果字段1相同，则按字段2排序，依次类推。 
例如:
	SELECT * FROM tn ORDER BY 字段1 DESC, 字段2 ASC;
3、ORDER BY中还可以用算法。 
例如:
	在论坛帖子中，我要按点击数和回复数的10倍的和排序列出所有帖子。 
	SELECT * FROM tn ORDER BY (点击数+回复数*10) DESC; 

二、聚合函数
SUM(), COUNT(), AVG()等都是聚合函数
常与GROUP BY配合使用

三、GROUP BY是按表中某些字段分类（分类汇总）
1、单个字段分类 
例如：按部门名称分类，统计各部门的职工人数和工资总额
SELECT dept_name, COUNT(id), SUM(salary) FROM employee GROUP BY dept_name;
2、多个字段分类
例如：按部门分类之后在按年龄分组。 
SELECT dept_name, age, COUNT(*) AS num FROM employee GROUP BY dept_name, age;
先以dept_name分组，相同，再根据age分组，num统计的是age分组条数！

四、HAVING条件，对分组后的数据过滤
例如：查询出员工平均薪资大于2000的部门
SELECT dept_name FROM employee GROUP BY dept_name HAVING AVG(salary) > 2000;

Note:
当使用GROUP BY分组时，使用SELECT *，则显示的数据为每组的第一条记录！
在sql命令格式使用的先后顺序上，GROUP BY先于ORDER BY！
